# IAAGS Internal Readiness Organizer

This repository contains a Streamlit app designed as an internal environmental readiness organizer for businesses in Aguascalientes and related supplier-preparation contexts.

It is intended to help structure internal observations around waste-management readiness, operational preparation, documentation, traceability, and supplier qualification considerations. It is not an official audit, legal opinion, or regulatory inspection.

## Features

- photo upload and evidence capture
- internal checklist for storage, transport, documentation, and response readiness
- risk-based report generation
- supplier-readiness framing for industrial and manufacturing contexts
- CSV export for internal recordkeeping

## Run locally

1. Install dependencies

   ```bash
   pip install -r requirements.txt
   ```

2. Start the app

   ```bash
   streamlit run streamlit_app.py
   ```

## Notes

- The app is intended for internal planning and preparation.
- It should be used alongside qualified environmental and regulatory guidance where applicable.
